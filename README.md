  ## Description
  ### This API was built using Express,js and Mongoose for a social network where users can share thoughts, reactions to thoughts and friends thoughts, and create friends lists. Thoughts are timestamped and all data can be updated or deleted using Insomnia

  ## Installation Instructions
  ### In the integrated terminal use npm i 

  ## Usage
  ### Clone the repo and install dependencies using npm i. Run npm server.js to begin and use Insomnia to test the API
